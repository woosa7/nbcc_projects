*** 1. cleaning up of the pdb file

I removed all the remarks and connectivity information from the pdb file.
I removed the chain B-J.
I removed all GOL and HOH molecules.
I removed ANISOU lines by typing

  cat cv30com_combined.pdb | gawk '{if ($1 != "ANISOU") print $0}' > cv30com_cleaned.pdb

The resulting pdb file is named "cv30com_cleaned.pdb".

*** 4. capping the terminal residues.

The special N- and C-terminal cap residues which can be used to neutralize the N- and C-terminal in peptide chains.

The "ACE" residue [-C(=O)-CH3] was used to cpa the N-terminal by using ALA residue which had been inserted with SWISS PDB Viewer.
The PDB entry of the capping residue ACE must be

   ATOM      2  CH3 ACE E 325     134.435 -12.116 -29.528  1.00 99.99
   ATOM      3  C   ACE E 325     133.387 -11.216 -29.966  1.00 99.99
   ATOM      4  O   ACE E 325     132.812 -10.491 -29.157  1.00 99.99


For capping the C-terminus, I used simple NH2 termination giving [C(=O)-NH-CH3]. 
This residue is called "NME" in the PDB file and consists of a single atom to be named N.

  ATOM   1648  N   NME E 533     136.519 -12.972 -29.721  1.00 99.99
  ATOM   1649  CH3 NME E 533     136.757 -11.543 -29.758  1.00 99.99

The resulting pdb file is named "cv30com_final.pdb".
